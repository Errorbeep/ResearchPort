# ResearchPort
科研管理系统
a basic research management system developed with Django

### 技术栈
+ Python3/Django
+ Javascript/CSS/HTML
+ sqlite

### 主要功能
+ 项目管理/论文管理
+ 用户/权限管理

### 运行
`python3 manage.py runserver`
